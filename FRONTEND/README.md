## 🚀 How to Start

### 1. Run the App

```bash
cd FRONTEND
npm run dev
```

Visit: **http://localhost:5173**

You should see a simple welcome message!

### 2. Read the Beginner's Guide

Check the **BEGINNER_GUIDE.md** artifact for:

- Step-by-step tutorials
- Code examples
- Learning resources
- Common mistakes to avoid

### 3. Start Building!

- Open `src/App.jsx` and start coding
- Create new components in `src/components/`
- Create new pages in `src/pages/`
- Use PrimeReact components from https://primereact.org/

## 📝 What's in Each File

### `src/App.jsx`

Simple starter component - **this is where you begin!**

### `src/components/layout/Header.jsx`

Empty header with TODO comment - **build your navigation here**

### `src/components/layout/Footer.jsx`

Empty footer with TODO comment - **add your footer content**

### `src/pages/HomePage.jsx`

Empty home page - **your landing page**

### `src/pages/ProductsPage.jsx`

Empty products page - **show your products here**

### `src/styles/theme.css`

PrimeReact theme imports - **already set up for you**

### `src/styles/global.css`

Basic CSS reset - **add your custom styles here**

## 💡 Learning Path

### Beginner (Week 1)

1. Create a ProductCard component
2. Display products on ProductsPage
3. Add Header and Footer to App
4. Style with PrimeReact components

### Intermediate (Week 2)

5. Add React Router for navigation
6. Create CartPage
7. Fetch data from backend API
8. Create login form

### Advanced (Week 3)

9. Implement shopping cart
10. Add authentication
11. Create admin pages
12. Deploy your app
